# Banking Data Engineering Case Study

This project demonstrates an end-to-end data engineering pipeline using
AWS, Databricks, PySpark, and SQL.

## Data
- Customers
- Accounts
- Transactions

## Tech Stack
- AWS S3 (Data Lake)
- Databricks (PySpark + SQL)
- GitHub (Version Control)

## Objectives
- Load and validate banking data
- Perform analytical queries using SQL and PySpark
- Identify data quality issues
- Explain orchestration approach
